Projet : Application Web Jakarta EE (JSP, Servlet, Hibernate)
Ce projet consiste à développer une application web complète en utilisant une architecture DAO générique pour assurer la persistance des données avec Hibernate.
🛠️ Stack Technique & Prérequis
Java : JDK 11 ou version supérieure.
Serveur : Apache Tomcat 10.x (Indispensable pour le namespace jakarta.*).
Build Tool : Maven.
Technologies : JSP, Servlets, Hibernate, Jakarta EE.
📝 Plan de Réalisation
1. Avant de commencer
Vérifiez que votre environnement est prêt :
java -version
mvn -version
Tomcat 10 configuré dans votre IDE (Eclipse/IntelliJ).
